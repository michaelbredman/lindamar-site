# Linda Mar Laundromat — Website

Official website for Linda Mar Laundromat, a self-service laundromat in Pacifica, CA.

**Live Site:** [lindamarlaundromat.com](https://www.lindamarlaundromat.com)

## About

- **Location:** 1349 Linda Mar Shopping Center, Pacifica, CA 94044
- **Phone:** (650) 514-5664
- **Hours:** Open 7 Days a Week, 6:00 AM – 10:00 PM
- **Equipment:** Dexter Laundry commercial washers (20–80 lb) and stack dryers (30–50 lb)
- **Payment:** LaundroWorks card-based system (no coins needed)

## Tech Stack

- Static HTML/CSS/JS (single page)
- Hosted on [Vercel](https://vercel.com)
- SEO optimized with JSON-LD structured data (LocalBusiness, FAQ, BreadcrumbList, WebSite schemas)

## Deployment

This site auto-deploys to Vercel on every push to `main`.

To deploy manually:
```bash
npx vercel --prod
```

## Local Development

Just open `index.html` in a browser — no build step required.
